чтоб б  файл(файлы) из папки проекта Pycharm попал в пепозиторий на гитхабе надо:

0.  начало 
- Зайти в нужный проект из программы Пайчарм c:\myproject\week2
- при необходимомти произвести настройки окружения venv (из моего видео) 
- создать новый пробный файл test_week2.py с любым содержимым , 
или выбрать  имеющийся файл и изменить в нем что-нибудь и сохранить .  



1.  ADD
 в Пайчарм выбрать один  конкретный файл (или весь  проект) 
выполнить команду гита  add    (после этого файл из красного станет зелёным)

2.  COMMIT
При нажатии commit текущее состояние файла(файлов) проекта сохранится в 
локальной  папке  c:\myproject\week2\.git

3.  PUSH
после выполнения  ПУШ  в репозитарий  на  гитхабе 
https://github.com/starichok2019/week2
этот файл test_week2.py(или файлы)  будет скопирован.


4.  Проверка
зайти в браузере в
https://github.com/starichok2019/week2
и увидеть этот файл и внесённые в него измения 


5.  ТОЛЬКО ПОСЛЕ выполнения п.4   можно далее кому-то  задавать вопросы
 по содержимому файла.    Или передавать ссылку на репозитарий(или файл) в
 ДЗ.
